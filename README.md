# Sales Data Analysis Dashboard

## 📊 Project Overview
This project analyzes sales data to identify trends, patterns, and insights using Excel and Python.

## 🔧 Tools Used
- Microsoft Excel
- Python (Pandas, Matplotlib)

## 📈 Key Insights
- Technology category has highest sales
- Office Supplies show consistent performance
- Furniture has lowest contribution

## 📂 Files
- sales_dashboard.xlsx → Excel dashboard
- sales_analysis_report.pdf → Project report

## 🚀 Goal
To build a strong foundation in Data Analysis and move towards Data Science and Machine Learning.
